## Overview
A promotional and e-commerce website for "Importadora Elefante", a clothing retailer in Panama. This project is built on WordPress to allow for easy content management and product updates by the client.

## Goals
- **Showcase Catalog**: Display the latest fashion collections.
- **Store Location**: Help customers find the physical store.
- **Brand Identity**: Establish a strong online presence for the brand.
