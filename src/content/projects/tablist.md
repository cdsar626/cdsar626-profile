## Overview
Tablist is a task management application designed to help users organize their daily activities. Currently available on the Google Play Store, it offers a clean interface and real-time syncing.

## Features
- **Task Management**: Add, edit, and complete tasks easily.
- **Cloud Sync**: Powered by MongoDB for data persistence across devices.
- **Flutter**: Cross-platform performance and native-like feel.
- **Dark Mode**: Built-in dark mode support.
