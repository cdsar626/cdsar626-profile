## Overview
A clone of Andreas Kitchen Meals website, re-engineered using modern web technologies. Instead of using Wix, this project integrates the Shoprocket store for e-commerce functionality.

## Tech Stack
- **Astro**: For high-performance static site generation.
- **SolidJS**: For fine-grained reactivity in interactive components.
- **TailwindCSS**: For rapid and consistent styling.
- **Shoprocket**: For handling the shopping cart and checkout process.
