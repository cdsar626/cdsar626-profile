# Portfolio Website

A sleek and modern portfolio website showcasing development projects and skills. Built with Astro for optimal performance and Vue.js for interactive components.

## Key Features

- **Project Showcase**: Interactive grid layout with hover effects
- **CV Viewer**: Integrated PDF viewer with smooth animations
- **Responsive Design**: Mobile-first approach with fluid layouts
- **Performance Optimized**: Static site generation with minimal JavaScript
- **Content Management**: Easy project management through markdown files
- **Smooth Animations**: CSS-based animations with reduced motion support

## Technical Implementation

The site leverages Astro's static site generation capabilities with Vue.js components for interactive elements. Tailwind CSS provides utility-first styling, while Content Collections manage project data.

## Challenges Solved

- Achieved 100% Lighthouse performance score
- Implemented accessible animations with motion preferences
- Created efficient content management workflow
- Built responsive design system with consistent spacing
